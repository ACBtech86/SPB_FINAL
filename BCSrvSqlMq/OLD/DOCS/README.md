# BCSrvSqlMq - Serviço de Integração SPB

## 🎯 Visão Geral

**BCSrvSqlMq** é um serviço Windows que integra sistemas do Sistema de Pagamentos Brasileiro (SPB) utilizando:
- **PostgreSQL** - Persistência de dados
- **IBM MQ** - Mensageria entre sistemas
- **Cryptlib** - Assinatura e criptografia
